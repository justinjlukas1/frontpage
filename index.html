Project Description

The true front page for the 21st century

FrontPage is a web application that provides a centralized location for users to access trending news. The application also supports searching for specific topics across a wide variety of social media platforms and news outlets and condenses the results based on popularity. This document outlines the requirement specifications for the implementation of FrontPage. As there is not currently one application that offers trending news from both social media sites and new sources in such a simplified format, FrontPage aims to become a centralized platform for users to access top news stories with ease, without having to visit and scroll through multiple websites and applications to access the most popular posts.
